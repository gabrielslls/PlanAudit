## 倍率计算方法

```
计费倍率 = 控制台显示的使用量 / 本地代理记录的实际请求数
```

| 倍率范围 | 判断标准 |
|----------|----------|
| 0.9 - 1.1 | 正常（技术误差范围内） |
| 1.1 - 1.2 | 需关注（可能存在技术性偏差） |
| > 1.2 | 异常（建议进一步核实） |
